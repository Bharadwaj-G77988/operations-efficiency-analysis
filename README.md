# Operations Efficiency & Performance Analysis

## Project Overview
This project analyzes operational performance and employee productivity using Python and Excel. The goal is to identify efficiency gaps, performance trends, and optimization opportunities.

## Tools Used
- Python (Pandas, Matplotlib)
- Excel
- Data Analysis & KPI Reporting

## Key Features
- Automated KPI calculations
- Department performance analysis
- Absence and overtime impact evaluation
- Excel dashboard with embedded charts

## Files Included
- Final Operations Dashboard (Excel)
- Dataset
- Charts
